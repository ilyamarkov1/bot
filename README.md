# VK-бот для вовлечения обучающихся в использование образовательного ресурса

Готовый учебный проект ВК-бота для сопровождения обучающихся при работе с образовательным ресурсом:

**Ресурс:** https://vk.ru/club237433614

## Идея проекта

Бот не просто выдает ссылку на ресурс, а сопровождает ученика по модулям:
- помогает выбрать конструктор;
- кратко объясняет цель и задание;
- предлагает мини-тест после изучения материала;
- собирает рефлексию;
- показывает личный прогресс и рекомендует следующий модуль.

Это позволяет обосновать, что бот **обеспечивает включенность обучающихся** в использование образовательного ресурса.

## Встроенные модули

1. Гексагон
2. Пазл-лабиринт
3. Зеркальный лабиринт
4. Кубический вызов
5. Деревянный замок

## Что делает бот

### Для обучающегося
- стартовое меню;
- выбор модуля;
- просмотр цели занятия;
- просмотр задания;
- мини-тест из 3 вопросов;
- рефлексия по модулю;
- просмотр личного прогресса.

### Для проекта / защиты
- показывает цифровое сопровождение образовательного ресурса;
- фиксирует прохождение модулей;
- демонстрирует интерактивность;
- повышает мотивацию за счет баллов и последовательного маршрута.

## Техническая реализация

- Python
- Flask
- VK Callback API
- requests
- SQLite

## Структура проекта

- `app.py` — основной файл бота
- `content.py` — контент модулей, вопросы, кнопки
- `storage.py` — база данных SQLite и прогресс пользователей
- `.env.example` — пример переменных окружения
- `requirements.txt` — зависимости

> ВК-бот является цифровым инструментом сопровождения образовательного ресурса. Он помогает обучающимся выбирать модуль, получать инструкции, проходить мини-проверку и фиксировать рефлексию. За счет этого повышается вовлеченность, системность работы с ресурсом и удобство взаимодействия с учебными материалами.
