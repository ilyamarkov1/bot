# VK-бот для вовлечения обучающихся в использование образовательного ресурса

Готовый учебный проект ВК-бота для сопровождения обучающихся при работе с образовательным ресурсом:

**Ресурс:** https://vk.ru/club23743361

## Идея проекта

Бот не просто выдает ссылку на ресурс, а сопровождает ученика по модулям:
- помогает выбрать конструктор;
- кратко объясняет цель и задание;
- предлагает мини-тест после изучения материала;
- собирает рефлексию;
- показывает личный прогресс и рекомендует следующий модуль.

Это позволяет обосновать, что бот **обеспечивает включенность обучающихся** в использование образовательного ресурса.

## Встроенные модули

1. Гексагон
2. Пазл-лабиринт
3. Зеркальный лабиринт
4. Кубический вызов
5. Деревянный замок

## Что делает бот

### Для обучающегося
- стартовое меню;
- выбор модуля;
- просмотр цели занятия;
- просмотр задания;
- мини-тест из 3 вопросов;
- рефлексия по модулю;
- просмотр личного прогресса.

### Для проекта / защиты
- показывает цифровое сопровождение образовательного ресурса;
- фиксирует прохождение модулей;
- демонстрирует интерактивность;
- повышает мотивацию за счет баллов и последовательного маршрута.

## Техническая реализация

- Python
- Flask
- VK Callback API
- requests
- SQLite

## Структура проекта

- `app.py` — основной файл бота
- `content.py` — контент модулей, вопросы, кнопки
- `storage.py` — база данных SQLite и прогресс пользователей
- `.env.example` — пример переменных окружения
- `requirements.txt` — зависимости

## Как запустить локально

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Создайте `.env`

Скопируйте `.env.example` в `.env` и заполните значения:

```env
VK_GROUP_TOKEN=ваш_токен_сообщества
VK_CONFIRMATION_TOKEN=строка_подтверждения
VK_SECRET_KEY=секретный_ключ_callback_api
VK_API_VERSION=5.199
PORT=8000
```

### 3. Запустите сервер

```bash
python app.py
```

## Как подключить к сообществу VK

1. Включите **Сообщения сообщества**.
2. Включите **Возможности ботов**.
3. Создайте **ключ доступа сообщества**.
4. В разделе **Callback API** укажите адрес вашего сервера.
5. Настройте **Secret key** и строку подтверждения.
6. В типах событий включите хотя бы `message_new`.

## Пример сценария для ученика

1. Открывает диалог с сообществом.
2. Нажимает «Модули».
3. Выбирает, например, «Гексагон».
4. Смотрит цель и задание.
5. Переходит к образовательному ресурсу.
6. Возвращается в бот и проходит мини-тест.
7. Пишет рефлексию.
8. Смотрит прогресс и получает рекомендацию по следующему модулю.
